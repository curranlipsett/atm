How to use?

Add to Makerlog let's you easily add a button to log tasks on Makerlog from your website. Think of it like a Tweet button but for Makerlog.

It can be used to log any task to Makerlog or as an easy, fast and no-code required implementation for Makerlog.

Add it to your website

It's very easy to add a button for your website, currently you have two options for buttons.

First create a class .atm-btn in your CSS and apply the suggested styles. 

.atm-btn {
  display: inline-block;
  padding: 5px 10px;
  background: #3BC089; /* Makerlog brand color */
  color: #FFFFFF;
  border-radius: 3px;
  text-transform: uppercase;
  text-decoration: none;
}

Add empty task button. It displays the input above to let the user type in any text he/she wants.
To do so simply create an anchor element in your website and point it to this page. 

<a href="https://atm.glitch.me" class="atm-btn" target="_blank">Add Task</a>

Try it Add Task

Add prefilled task button. It prefills the input above with your desired text, for example: "Added my first task with #AddToMakerlog".

To prefill a task input just append ?task={edit-text} at the end of the URL. {edit-text} must be encoded in the URL.
*Please note that users are free to modify the text using the input above. 

<a href="https://atm.now.sh/?task=Added%20my%20first%20task%20with%20%23AddToMakerlog" class="atm-btn" target="_blank">Add Task</a>

Try it Add Task

No-code option

If you don't want to use anchors or any piece of code, just create a link to the following URLs. 

https://atm.now.sh (empty task input).
https://atm.now.sh/?task=Desired text (prefilled task input).
